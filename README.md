# Include Users API

## Pre-requisites

- Node 16+
- Docker

## Get Started

- Install: `npm install`

- Build: `npm run build`

### Run tests

- Run tests: `npm run test`

### Run API localy

- Run docker compose to start a PostgreSQL Database: `docker compose up`

- Fill .env file according to .env.example

- Start server: `npm run start`
